# 先進的アニメーション強化 - 設計仕様

## 概要

UI再設計の第2弾。Three.js / R3F による3Dヒーロー背景、サイバーパンク風ローディング画面、グリッチ/ネオン/スキャンラインエフェクトを全ページに適用し、近未来感を最大化する。

## 設計方針

- **全部盛り**: 3D、グリッチ、ネオン、スキャンライン、ローディング演出を妥協なく実装
- **常時バリバリ**: テキストグリッチ常時、スキャンライン常時、ネオン点滅、ホバーでさらに強化
- **ローディング画面**: 3段階演出（ブートシーケンス → パーティクル集合ロゴ → グリッチ画面割れ）
- **`prefers-reduced-motion`**: 全エフェクト無効化（アクセシビリティ必須）

## 新規パッケージ

| パッケージ | 用途 |
|-----------|------|
| `three` | 3D レンダリング |
| `@react-three/fiber` | React 用 Three.js バインディング |
| `@react-three/drei` | R3F ヘルパー（Stars, Float, MeshDistortMaterial 等） |

## コンポーネント設計

### 1. ローディング画面 (`loading-screen.tsx`)

初回アクセス時に表示（sessionStorage で制御、セッション中は1回のみ）。

**Phase 1 (0-2s): ブートシーケンス**
- 黒背景にターミナル風テキストが高速で流れる
- `> INITIALIZING HOLOLIVE SYSTEM...`
- `> LOADING MEMBER DATABASE... 100+ ENTRIES`
- `> CONNECTING TO FAN NETWORK...`
- `> SYSTEM READY.`
- 緑色のモノスペースフォント、タイプライター表示

**Phase 2 (2-4s): パーティクル集合ロゴ**
- 散らばった光の粒子が中央に集合
- 「ホロライブ診断」のテキスト形状を形成
- ネオンシアングロウ (#00B8ED)

**Phase 3 (4-5s): グリッチ画面割れ**
- ロゴがグリッチエフェクト（RGB ずれ、スキュー）
- 画面がスライスされて割れる演出
- 本編にトランジション

**スキップ**: 画面タップ / クリックで即座にスキップ

### 2. 3D ヒーロー背景 (`hero-3d-scene.tsx`)

React Three Fiber による3Dシーン。

- **メッシュグラデーション球体**: `MeshDistortMaterial` で有機的にうねる球体
- **星空パーティクル**: `@react-three/drei` の `Stars` コンポーネント
- **浮遊する幾何学オブジェクト**: 小さな正二十面体 / トーラスが `Float` でゆらゆら浮遊
- **カラー**: ホロライブブルー (#00B8ED) + パープル (#6C5CE7) のグラデーション
- **マウス追従**: カメラがマウス位置に微妙に追従（パララックス感）
- `dynamic(() => import(...), { ssr: false })` で読み込み

### 3. グリッチテキスト (`glitch-text.tsx`)

CSS アニメーションによるグリッチエフェクト。

- `::before` / `::after` 疑似要素で RGB ずれ（赤・シアン）
- ランダムな `clip-path` アニメーション
- 常時軽度のグリッチ + ホバーで強度UP
- テキストカラーにネオングロウ (`text-shadow`)

### 4. ネオングロウテキスト (`neon-text.tsx`)

ネオンサイン風のグロウエフェクト。

- `text-shadow` の多重指定で発光感
- 微妙なフリッカー（点滅）アニメーション
- カラーカスタマイズ可能（デフォルト: #00B8ED）

### 5. スキャンライン (`scanlines.tsx`)

CRT モニター風のスキャンラインオーバーレイ。

- `::after` 疑似要素で `repeating-linear-gradient`
- 半透明（opacity: 0.03-0.05）で全画面を覆う
- 微妙にスクロールするアニメーション
- レイアウトに `position: relative` でラップ

### 6. サイバーグリッドフッター (`cyber-grid.tsx`)

フッター背景のパース付きグリッド。

- CSS `perspective` + `rotateX` でグリッドが奥に消えていく
- グリッド線がネオンカラーで発光
- アニメーションでグリッドが手前に流れる

### 7. ページ遷移強化 (`cyber-page-transition.tsx`)

既存の `page-transition.tsx` を拡張。

- 遷移時にグリッチオーバーレイが走る（100ms）
- RGB ずれ + ノイズ → フェード → 新ページ
- スキャンラインが遷移中に強くなる

## 適用先

| ページ | 追加エフェクト |
|--------|-------------|
| **全ページ** | スキャンラインオーバーレイ、ネオングロウヘッダー、サイバーページ遷移 |
| **ローディング** | 3段階ローディング画面（初回のみ） |
| **トップページ** | 3D ヒーロー背景、グリッチタイトル、ネオンCTAボタン |
| **メンバーハブ** | グリッチセクション見出し |
| **世代一覧** | ネオン世代名、ヒーローバナーにスキャンライン強化 |
| **メンバー詳細** | ネオン名前表示 |
| **診断** | グリッチ質問遷移 |
| **結果** | グリッチ演出強化、ネオンスコア表示 |
| **フッター** | サイバーグリッド背景 |

## パフォーマンス

- Three.js シーンは `dynamic(() => import(...), { ssr: false })` で遅延読み込み
- モバイル: 3D シーンのポリゴン数を削減、Star count を半減
- `prefers-reduced-motion`: 全アニメーション無効、グリッチ/スキャンライン非表示
- グリッチ CSS は `will-change: transform` で GPU レンダリング
